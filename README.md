# 「建築・デザイン事務所」中心のネットワーク調査

建築家・デザイナーの師弟関係・系譜をネットワーク図として可視化するWebアプリケーションです。
CSVファイルをドラッグ＆ドロップするだけで、師弟関係のグラフを即座に描画できます。

**デモ**: [https://shinobu-u.github.io/design-network/](https://shinobu-u.github.io/design-network/)

---

## 主な機能

- CSVファイルのドラッグ＆ドロップによるデータ読み込み
- 師弟関係のネットワーク図（Cytoscape.js）
- 専門・国籍・所属事務所・出身校・設立事務所・留学歴によるカラーモード切り替え
- コミュニティグルーピング（同じ事務所・学校・専門・国籍でまとめて表示）
- 複数レイアウト対応（Force-directed / Breadth-first / Circle / Grid / Concentric / Random）
- 特定人物を中心にした放射状ネットワーク表示
- フィールドごとのキーワード絞り込みフィルター（OR検索）
- 年表表示（在籍年・生没年のガントチャート）
- ネットワーク特徴量の自動計算（次数、クラスター係数、モジュラリティ等）
- 次数分布チャート
- データのインライン編集・新規追加
- 書き出し：生データCSV / 匿名化データCSV / ネットワーク図（PNG・JPEG・SVG）
- 具体名の一括非表示（個人情報保護）
- 日本語 / English 切り替え

## 使い方

### 1. ページを開く

[デモページ](https://shinobu-u.github.io/design-network/) をブラウザで開きます。
ローカルで試す場合は `index.html` をブラウザで直接開いてください（`style.css` / `app.js` と同じディレクトリに置く必要があります）。

### 2. データを読み込む

- 付属の `sample_data.csv` をドラッグ＆ドロップして動作を確認できます。
- 独自のCSVを用意してドロップすることで、任意のネットワークを可視化できます。

### 3. 表示を調整する

| 操作 | 説明 |
|---|---|
| カラーモード | ノードの色分け基準を変更（専門 / 国籍 / 所属事務所 / 出身校 等） |
| コミュニティ | 同じ属性のノードをまとめて表示 |
| レイアウト | グラフの配置方法を変更 |
| 絞り込み | フィールドごとにキーワードでノードを絞り込み（カンマ区切りで複数指定可・OR検索） |
| ネットワークを更新 | データ編集後にグラフを再描画（ネットワーク図左上） |
| 具体名を非表示 | 氏名・学校・事務所名をアルファベット記号に置換（ネットワーク図右下） |

## CSVフォーマット

文字コードは **UTF-8**、1行目はヘッダーとして読み飛ばされます。

```
識別番号,氏名,専門,国籍,師匠(複数は|区切り),出身校,留学歴,所属事務所(複数は|区切り),生没年,在籍年(複数は|区切り),独立年,設立事務所,備考
```

| フィールド | 内容 | 例 |
|---|---|---|
| 識別番号 | 整数（一意） | `0` |
| 氏名 | フルネーム | `山田花子` |
| 専門 | `0`=建築 / `1`=インテリア / `2`=グラフィック / `3`=プロダクト / `4`=写真 | `0` |
| 国籍 | 国籍（空欄可） | `日本` |
| 師匠 | 師匠の氏名（複数は `\|` 区切り） | `田中一郎\|鈴木次郎` |
| 出身校 | 学校名 | `○○大学` |
| 留学歴 | 留学先（空欄可） | `△△工科大学` |
| 所属事務所 | 在籍した事務所名（複数は `\|` 区切り） | `田中建築事務所\|鈴木アトリエ` |
| 生没年 | `YYYY-YYYY` または `YYYY-` | `1960-` |
| 在籍年 | 期間（複数は `\|` 区切り） | `1985–1991\|1992–1998` |
| 独立年 | 独立した年 | `1999` |
| 設立事務所 | 自身が設立した事務所名 | `山田設計事務所` |
| 備考 | 自由記述 | `コンクリート打放し建築` |

### 記入例

```csv
識別番号,氏名,専門(建築:0/インテリア:1/グラフィック:2/プロダクト:3/写真:4),国籍,師匠(複数は|区切り),出身校,留学歴,所属事務所(複数は|区切り),生没年,在籍年(複数は|区切り),独立年,設立事務所,備考
0,田中一郎,0,日本,,○○大学,,田中建築事務所,1930-2010,,1960,田中アトリエ,
1,鈴木次郎,0,日本,田中一郎,△△大学,,田中建築事務所,1955-,1980–1988,1989,鈴木設計室,
2,山田花子,1,日本,田中一郎|鈴木次郎,□□大学,◇◇工科大学,鈴木設計室,1965-,1990–1997,1998,山田スタジオ,2名に師事した例
```
