# 建築・デザイン師弟ネットワーク

建築家・デザイナーの師弟関係・系譜をネットワーク図として可視化するWebアプリケーションです。
CSVファイルをドラッグ＆ドロップするだけで、師弟関係のグラフを即座に描画できます。

**デモ**: [https://shinobu-u.github.io/design-network/](https://shinobu-u.github.io/design-network/)

---

## 主な機能

- CSVファイルのドラッグ＆ドロップによるデータ読み込み
- 師弟関係のネットワーク図（Cytoscape.js）
- 専門・所属事務所・出身校・設立事務所・留学歴によるカラーモード切り替え
- コミュニティグルーピング（同じ事務所・学校・専門でまとめて表示）
- 複数レイアウト対応（Force-directed / Breadth-first / Circle / Grid / Random）
- ネットワーク特徴量の自動計算（次数、クラスター係数、モジュラリティ等）
- 次数分布チャート
- データのインライン編集・新規追加
- CSVダウンロード・具体名非表示CSVダウンロード
- 具体名の一括非表示（個人情報保護）
- 日本語 / English 切り替え

## 使い方

### 1. ページを開く

[デモページ](https://shinobu-u.github.io/design-network/) をブラウザで開きます。
ローカルで試す場合は `index.html` をブラウザで直接開いてください。

### 2. データを読み込む

- 付属の `sample_data.csv` をドラッグ＆ドロップして動作を確認できます。
- 独自のCSVを用意してドロップすることで、任意のネットワークを可視化できます。

### 3. 表示を調整する

| 操作 | 説明 |
|---|---|
| カラーモード | ノードの色分け基準を変更（専門 / 所属事務所 / 出身校 等） |
| コミュニティ | 同じ属性のノードをまとめて表示 |
| レイアウト | グラフの配置方法を変更 |
| 具体名を非表示 | 氏名・学校・事務所名をアルファベット記号に置換 |

## CSVフォーマット

文字コードは **UTF-8**、1行目はヘッダーとして読み飛ばされます。

```
識別番号,氏名,指導教授(複数は|区切り),出身校,留学歴,所属事務所(複数は|区切り),専門,生没年,在籍年(複数は|区切り),独立年,設立事務所,備考
```

| フィールド | 内容 | 例 |
|---|---|---|
| 識別番号 | 整数（一意） | `0` |
| 氏名 | フルネーム | `山田花子` |
| 指導教授 | 師匠の氏名（複数は `\|` 区切り） | `田中一郎\|鈴木次郎` |
| 出身校 | 学校名 | `○○大学` |
| 留学歴 | 留学先（空欄可） | `△△工科大学` |
| 所属事務所 | 在籍した事務所名（複数は `\|` 区切り） | `田中建築事務所\|鈴木アトリエ` |
| 専門 | `0`=建築 / `1`=インテリア / `2`=グラフィック / `3`=プロダクト / `4`=ファッション | `0` |
| 生没年 | `YYYY-YYYY` または `YYYY-` | `1960-` |
| 在籍年 | 期間（複数は `\|` 区切り） | `1985–1991\|1992–1998` |
| 独立年 | 独立した年 | `1999` |
| 設立事務所 | 自身が設立した事務所名 | `山田設計事務所` |
| 備考 | 自由記述 | `コンクリート打放し建築` |

### 記入例

```csv
識別番号,氏名,指導教授(複数は|区切り),出身校,留学歴,所属事務所(複数は|区切り),専門(建築:0/インテリア:1/グラフィック:2/プロダクト:3/ファッション:4),生没年,在籍年(複数は|区切り),独立年,設立事務所,備考
0,田中一郎,,○○大学,,田中建築事務所,0,1930-2010,,1960,田中アトリエ,
1,鈴木次郎,田中一郎,△△大学,,田中建築事務所,0,1955-,1980–1988,1989,鈴木設計室,
2,山田花子,田中一郎|鈴木次郎,□□大学,◇◇工科大学,鈴木設計室,1,1965-,1990–1997,1998,山田スタジオ,2名に師事した例
```

## 使用ライブラリ

| ライブラリ | バージョン | 用途 |
|---|---|---|
| [Cytoscape.js](https://js.cytoscape.org/) | 3.28.1 | ネットワーク可視化 |

| [KaTeX](https://katex.org/) | 0.16.9 | 数式レンダリング |
| [Google Fonts](https://fonts.google.com/) | — | Chivo / Inter |
